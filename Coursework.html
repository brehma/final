Reflection on the Class:
Well I went into this class with ZERO knowledge of HTML, CSS or JavaScript.   What did help is all my previous programming experience in C and C++.   But that did nada for the basic HTML and CSS stuff.  Thanks to this class, I gained what I think is a reasonable base of knowledge of HTML, CSS and JavaScript.   However, what may be even more important, is that I also gained knowledge of reference resources I can turn to for increasing my knowlege or when I have a specific question in mind.   All in all I'm looking forward to next quarter and more Web programming.
